# Resources — Profile likelihood

Verified external explanations, if you want another angle. (Every link below was checked this
session.)

- 📖 [Wikipedia: Likelihood function → Profile likelihood](https://en.wikipedia.org/wiki/Likelihood_function)
  (reference) — the "Profile likelihood" section describes exactly the move this lesson builds:
  "slicing the likelihood surface along the ridge of values of the nuisance parameter … that
  maximizes the likelihood," reducing it to a function of the parameter of interest. Read that
  one section (the ridge-slicing picture matches the lesson's mountain-ridge analogy).
- 📖 [Wikipedia: Chi-squared distribution (Definitions)](https://en.wikipedia.org/wiki/Chi-squared_distribution)
  (reference) — for the degrees-of-freedom half of this lesson: it confirms a $\chi^2_k$ is the
  sum of $k$ independent squared normals, which is why profiling out one parameter (using up one
  of those pieces) drops the df by one. Skim the lead and "Definitions."
