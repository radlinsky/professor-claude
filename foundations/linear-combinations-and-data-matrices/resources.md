# Resources: Linear combinations & the data matrix

All links verified 2026-07-04.

- 📺 [3Blue1Brown — Linear combinations, span, and basis vectors (Essence of Linear Algebra, Ch. 2)](https://www.youtube.com/watch?v=k7RM-ot2NWY)
  (video, ~10 min) — the clearest picture of "scale each vector and add them up" — a
  linear combination — animated. Watch after this lesson's "Getting a feel for it"
  section to see the multiply-and-add move geometrically; ignore the "span / basis"
  vocabulary for now, the linear-combination half is what you need.
- 🎮 [3Blue1Brown — Linear combinations, span, and basis vectors (written + interactive lesson)](https://www.3blue1brown.com/lessons/span)
  (interactive article) — the same chapter as a text lesson with diagrams and quick
  comprehension checkpoints; good if you'd rather read and click than watch. Reinforces
  that a linear combination is just "scale and add."
- 📺 [StatQuest — One-Hot, Label, Target and K-Fold Target Encoding, Clearly Explained](https://www.youtube.com/watch?v=589nCGeWG1w)
  (video, ~34 min) — Josh Starmer's plain-English walk through turning a category into
  0/1 columns (one-hot encoding), with tiny examples. Watch the first ~10 minutes
  (one-hot encoding) after this lesson's indicator-column section; it's exactly the
  `tierB` / `tierC` columns we built, from another angle. The later target/k-fold
  encoding parts go beyond this module — optional.
