# Resources: Decision trees

All links verified 2026-07-12.

- 📺 [StatQuest: Decision Trees](https://www.youtube.com/watch?v=7VeUPuFGJHk)
  (video) --- Josh Starmer builds a classification tree step by step, showing how
  Gini impurity picks the best split. Watch after the lesson to see the same greedy
  algorithm animated with different visuals.
- 📖 [ISLR §8.1 — The Basics of Decision Trees](https://hastie.su.domains/ISLR2/ISLRv2_corrected_June_2023.pdf)
  (book, PDF) --- the textbook treatment this module follows: regression trees
  (§8.1.1--8.1.2), classification trees (§8.1.3), trees vs. linear models (§8.1.4),
  and the advantages/disadvantages summary (§8.1.5). Read after the lesson for the
  formal algorithm and the Hitters data example.
- 📺 [StatQuest: Pruning Regression Trees](https://www.youtube.com/watch?v=D0efHEJsfHo)
  (video) --- focuses on cost-complexity pruning and cross-validation for selecting
  the tree size. Watch after the pruning section for a visual walkthrough of the
  grow-then-prune approach.
