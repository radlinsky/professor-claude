# Resources: How wrong is the model, really?

All links verified 2026-07-04.

- 📺 [StatQuest — R-squared, Clearly Explained!!!](https://www.youtube.com/watch?v=2AQKmw14mHM)
  (video, ~11 min) — walks R² with tiny numeric examples in plain English: "the
  percentage of variation the model accounts for," which is exactly this lesson's
  $1 - \text{RSS}/\text{TSS}$. Watch after the "R² — better than guessing the mean?"
  section to cement the intuition.
- 📺 [StatQuest — Machine Learning Fundamentals: Cross Validation](https://www.youtube.com/watch?v=fSytzGwwBVw)
  (video, ~6 min) — the k-fold picture animated: chop the data into folds, rotate which
  fold is held out, average the scores. Watch alongside this lesson's cross-validation
  loop to see the rotation move before you read the R.
- 📺 [StatQuest — Machine Learning Fundamentals: Bias and Variance](https://www.youtube.com/watch?v=EuBBz3bI-aA)
  (video, ~6 min) — the clearest short explanation of the bias–variance trade-off, with
  the under-/over-fitting curves this lesson keeps in words. Watch after the
  "bias–variance trade-off" section to put pictures to the two failure modes.
