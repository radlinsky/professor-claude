# Resources: Differentiation rules & recipes

Verified external explanations for this foundation module. Watch/read alongside the
lesson if a second voice helps.

- 📺 **[3Blue1Brown — Derivative formulas through geometry (Essence of Calculus, Ch. 3)](https://www.youtube.com/watch?v=S0_qX4VJhMQ)**
  (video, ~18 min). Derives the power rule visually — shows *why* "$nx^{n-1}$" falls out
  of geometry rather than just handing you the recipe. The best companion to the power-rule
  section; watch it if "bring the exponent down" feels like a magic trick without a
  reason.

- 📺 **[3Blue1Brown — Chain rule and product rule (Essence of Calculus, Ch. 4)](https://www.youtube.com/watch?v=YG15m2VwSjA)**
  (video, ~16 min). Builds the chain rule visually from the same "tiny nudge" picture this
  lesson uses for every rule. Directly extends the "outer × inner" intuition developed here.
  Especially useful if the gear-train picture of the chain rule hasn't clicked yet.

- 📺 **[Khan Academy — Basic differentiation rules](https://www.khanacademy.org/math/ap-calculus-ab/ab-differentiation-1-new)**
  (videos + exercises, self-paced). Covers the same five rules (constant, power,
  constant-multiple, sum/difference, chain) with worked examples and auto-graded practice
  problems. Useful for extra drill if the practice set above left any rule feeling shaky.
