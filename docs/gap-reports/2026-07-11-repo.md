## Reviewed, not findings

- **PCA "basis vectors are interpretable features" trap:** The lesson's Trap 1 already covers this territory ("PCA gives you the most important VARIABLES" addresses the same confusion about loading interpretation). The KB version adds alternatives (NMF, sparse PCA) but at this learner's level the existing trap is adequate.
- **PCA "always reduces to 2D without loss" trap:** The lesson teaches scree plots and PVE which implicitly address this. Explicitly trapping it would be redundant.
- **PCA "zero eigenvalue means error" and "no objective rule for components":** Both adequately addressed by lesson content (eigenvalue discussion and scree plot section).
- **PCA sign-flip uniqueness:** Not a practical trap for this learner who uses `prcomp()` and won't compare across implementations.
- **Clustering "empty groups can appear" (Boyd p. 75):** Edge case unlikely to confuse this learner on first encounter with k-means. The lesson uses `kmeans()` in R which handles this silently.
- **Clustering "visual clustering is misleading" in high dimensions:** Minor; the lesson's 2D toy is appropriate for first encounter and doesn't claim real data is 2D.
- **Distance "orthogonal means perpendicular" zero-vector edge case:** Too pedantic for this level; the lesson correctly introduces orthogonality as dot-product-equals-zero.
- **kNN Bayes classifier framing:** The KB records the Bayes error rate as the gold standard. Beyond this learner's current prerequisites and not a gap in what the kNN lesson teaches.
- **Covariance 1/N vs 1/(n-1) convention:** Austin uses 1/N (population); the PCA lesson uses 1/(n-1) (sample, matching R's `cov()`). The KB norm-and-distance page acknowledges this convention difference. Not a contradiction.
- **MVN distribution:** No significant gaps found between the multivariate-normal-distribution lesson and KB concept pages. The lesson's two traps (marginal normality ≠ joint normality; positive definiteness requirement) match the KB well.
- **Distance-similarity-and-geometry:** No significant gaps beyond the minor edge cases noted above. The lesson covers Euclidean distance, norms, Manhattan distance, correlation distance, projection, and orthogonality, all matching KB content.
